Lead Enquiry

Configure a default membership and staff for the new leads being generated from core, widget, apps or other websites.

Default Staff (Assign to)

Select a default staff that would be assigned to all new leads generated from core, widget, app and from the default mailbox. You can also select ‘Not Specified’ if you would like to assign a staff after the lead has come in.

Default MailBox

This email address can be linked to your website and other online forms for lead submission. For more information on how to configure this, please refer to our support platform.

Default Membership

Select a default membership that would be assigned to all new leads generated from the default mailbox, i.e. if you add the default mailbox to your website, all new leads generated from the website will have this default membership assigned to them.
