Communications

Suspend/unsuspend different communication means with your staff and customers.

Suspend Outgoing Emails

Toggle on to suspend all system generated and manual emails except password reset emails.

Display Name (optional)

A customized name that appears as a sender’s name in the recipient’s inbox.

‘Reply to’ Email Address (optional)

An email address that receives all email responses from your customers.

Suspend Outgoing SMSs

Toggle on to suspend all system generated and manual SMSs.

Suspend Push Notifications

Toggle on to suspend all system generated and manual push notifications.

Zero Value Invoices

Toggle on to suspend all outgoing emails for invoices where total amount due is zero.

Membership Benefits

Suspend/unsuspend membership benefits when a membership payment has failed. This setting does not override the ‘Enforce Manual Settings’ check in a member’s profile.

Suspend All Benefits

Toggle on to automatically suspend all or selective membership benefits when membership payment has failed for the respective interval.

Suspend Door Check-ins

Suspend Service Benefits

Suspend Class Benefits

Suspend Products Benefits

Show Prices Online – Widget & Customer App

Toggle on to show prices for items within a members’ membership that are currently unavailable due to item level configuration that are hiding their prices. E.g. If a Zumba class is currently configured to hide its prices online, but is part of a members’ membership, the member will not be able to book the Zumba class online unless this toggle is turned on.

Member Attendance

Set the default timer on the member attendance portal. Leaving the field blank or setting it to 0 would result in the portal never logging out the customer’s profile.

Access Control

Members

Toggle on to allow members to use access control systems in your branch.

Bluetooth Access

Toggle on to allow members to use their customer app to access the branch.

NFC Access

Toggle on to allow members to use their customer app or fobs to access the branch through NFC . Please note NFC access does not work with iOS devices.

Staff

Toggle on to allow staff to use access control systems in your branch.

Access Control – Timing

Set the default times for staff. Staff can either gain access anytime or during branch opening hours. You can also set a grace period for branch opening hours between 0 minutes to 60 minutes.

Bluetooth Access

Toggle on to allow staff to use their customer app to access the branch.

NFC Access

Toggle on to allow staff to use their customer app or fobs to access the branch through NFC . Please note NFC access does not work with iOS devices.

Class Configurations

Show Free Spaces - Widget & App

Toggle on to allow customers to see available spaces against each class on the widget & app.

Booking for Family and Friends

Toggle on to allow a customer to book multiple spaces in a class for family and friends.

Use Discounted Benefits for Booking Family and Friends

Toggle on to allow members to use their benefits for booking family and friends in the discounted classes.

Use Free Benefits for Booking Family and Friends

Toggle on to allow members to use their benefits for booking family and friends in the free classes.

Customer Check-In

Toggle on to allow customer to check-in for classes from the widget & app.

Check-In Starts Before

How early can a customer check-in before the class start time.

Check-In Ends After

How late can a customer check-in after the class start time. Selecting ‘Class Start Time’ would result in customers not being allowed to check in after the ‘Class Start Time’.

Date Filter for Classes

Set a default date filter for classes tab on widget & app.

Service Configurations

Customer Check-In

Toggle on to allow customer to check-in for services from the widget & app.

Check-In Starts Before

How early can a customer check-in before the service start time.

Check-In Ends After

How late can a customer check-in after the service start time. Selecting ‘Service Start Time’ would result in customers not being allowed to check in after the ‘Service Start Time’.

Service Lead Time

Select how close to a service time the customer can book with your branch from the widget & app.

Service Interval

Select the interval between time slots as they appear in core, widget & app.

Reward Program

Reward Program - Widget & App

Toggle on to hide ‘reward program menu’ from widget & app for customers if you wish to discontinue reward program for your branch. The configuration does not automatically terminate/freeze program for customers.

Cancellations and Refunds

Toggle on to automatically revoke all earned reward points when a booking is cancelled/marked as no show or an invoice is refunded.
